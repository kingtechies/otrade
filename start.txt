# OTRADE - Ultra Advanced AI Trading Bot

## Quick Start

### 1. Setup Virtual Environment
```bash
cd /home/jeph/otrade
source venv/bin/activate
```

### 2. Configure Credentials
```bash
cp .env.example .env
nano .env
```

Fill in:
- MT5_LOGIN (your Exness account number)
- MT5_PASSWORD (your password)
- MT5_SERVER (Exness-MT5Real or Exness-MT5Trial)
- OPENAI_API_KEY (your OpenAI key)

### 3. Start Wine MT5 Server
In a separate terminal:
```bash
# Install MT5 via Wine first, then:
WINEPREFIX=~/.wine64 wine64 ~/.wine64/drive_c/Program\ Files/MetaTrader\ 5/terminal64.exe

# Start Python bridge in Wine:
WINEPREFIX=~/.wine64 wine64 python -c "
import rpyc
from rpyc.utils.server import ThreadedServer
from rpyc import SlaveService
s = ThreadedServer(SlaveService, port=18812)
s.start()
"
```

### 4. Run the Bot
```bash
python main.py
```
